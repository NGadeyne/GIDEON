<h2>Intallation</h2>

Clone the repo.
```
git clone https://github.com/NGadeyne/GIDEON.git
```

Install packages.
```
pip install -r requirements.txt
```

Add .env file.
</br>Contents of .env file:

```
USER=None
BOTNAME=GIDEON
EMAIL=None
PASSWORD=None
NEWS_API_KEY=None
OPENWEATHER_APP_ID=None
TMDB_API_KEY=None
```

Replace `None` with your values
